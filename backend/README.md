# IMAGIN.AI Backend

API FastAPI pour le traitement d'images avec IA

##  Description
Backend pour IMAGIN.AI - Trois modèles d'IA basés sur l'image :
- **ARTIFY** : Transformation artistique (Neural Style Transfer)
- **INSIGHT** : Analyse d'émotions faciales
- **STYLE ME** : Recommandation vestimentaire
